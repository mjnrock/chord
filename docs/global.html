PixelScaleCanvas()

This will take a canvas and convert it into a larger canvas with [ width, height ] = [ width * @factor, height * @factor ]. It will then re-write each pixel to scale it to the new canvas. As such, this function explicitly creates this new canvas. Example: If @factor is 3, then each pixel would be re-written as a 3x3-pixel rectangle, and the canvas would be 3x as large, maintaining the original aspect ratio. NOTE: This is most useful for scaling bit-level images, such as 32x32 sprites to maintain the "look" of pixel art, but without the blurring as you zoom.
