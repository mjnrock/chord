Class: DataTable

DataTable()

Within the ETL framework, a DataTable is a place to store data. It is a Node, thus giving a Flux-like architecture to the data management and a rich event system for both effects and event systems. For example, a Cat data-table might could be automatically updated when an Animal data-table is updated, and the Cat data-table could have an effect that meows when the Cat data-table is updated. By combining the power of the Node architecture with the power of the DataTable, you can create a powerful data management system, in an extremely decoupled and modular way. Additionally, since the methods are defined using the flux system, you can easily extend/listen to the DataTable using the effect broadcasting.
